# TalapLab

AI-генератор корпоративных SCORM-курсов из ваших регламентов и инструкций. Загрузите PDF — получите готовый обучающий курс с тестом за час, а не за месяцы.

**Талап** (қаз. талап) — требование, регламент, устремление. **Lab** — лаборатория, где документы превращаются в учебные материалы.

**MVP-материалы:**

- **[Лендинг](index.html)** — маркетинговая посадочная страница
- **[Карта системы](sitemap.html)** — sitemap всех экранов MVP
- **[Прототип приложения](app.html)** — кликабельный фронт-демо (без бэкенда, состояние в localStorage)
- **[BRD](BRD_AI_CourseGen_MVP.md)** — функциональные требования MVP

## Стек прототипа

Чистый HTML/CSS/JS · без сборки · откройте `app.html` в браузере. Шрифты: Playfair Display + Montserrat.
