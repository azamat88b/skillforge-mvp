# SkillForge

AI-генератор корпоративных SCORM-курсов из документов. Загрузите регламент или инструкцию — получите готовый обучающий курс с тестом.

**MVP-материалы:**

- **[Лендинг](index.html)** — маркетинговая посадочная страница
- **[Карта системы](sitemap.html)** — sitemap всех экранов MVP
- **[Прототип приложения](app.html)** — кликабельный фронт-демо (без бэкенда, состояние в localStorage)
- **[BRD](BRD_AI_CourseGen_MVP.md)** — функциональные требования MVP

## Открыть демо

После деплоя через GitHub Pages — все три HTML-страницы доступны напрямую по URL.

## Стек прототипа

Чистый HTML/CSS/JS · без сборки · open `app.html` в браузере. Шрифты: Playfair Display + Montserrat.
